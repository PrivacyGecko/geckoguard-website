Frequently Asked Questions

Everything you need to know about Gecko Guard

Getting Started

Is this really free?

Yes! 100% free forever. Gecko Guard is open-source software with all privacy features available at no cost. No subscriptions, no hidden fees, no premium tiers. We're committed to keeping privacy protection accessible to everyone.

Is this open source?

Yes! Gecko Guard is fully open-source under the MIT license. You can review the code, contribute improvements, or fork the project on GitHub.

Can I provide feedback?

Absolutely! We welcome community input:

  • Discord: Join our community for discussions
  • GitHub Issues: Submit bug reports and feature requests
  • Email: contact@privacygecko.com for general suggestions

Privacy & Security

Do you collect my browsing data?

No. We don't collect, store, or transmit any of your browsing data. All processing happens locally on your device.

Why does the extension need so many permissions?

Each permission is essential for privacy protection:

  • activeTab: Read current page to apply protections
  • declarativeNetRequest: Block unwanted network requests
  • storage: Save your preferences locally
  • scripting: Inject privacy protection code

Is my data secure?

Yes. Since we don't collect your data, there's nothing to breach. All settings are stored locally on your device using browser security standards.

Can I see what's being blocked?

Yes! Click the extension icon to see real-time blocking statistics and detailed information about what's being protected.

Features

What does the extension actually block?

  • Trackers: Analytics, advertising, and social media trackers
  • Fingerprinting: Canvas, WebGL, audio, and device fingerprinting
  • Cookies: Manages and categorizes cookies with smart deletion
  • Social widgets: Facebook, Twitter, and other tracking widgets

Does it block ads?

The extension focuses on privacy protection, not ad blocking. It blocks tracking related to ads but may not block ad display.

Will it slow down my browsing?

No. The extension is optimized for minimal performance impact. Most users see improved page loading due to blocked content.

Can I customize the settings?

Yes! Access settings through the extension popup. You can adjust blocking levels, whitelist sites, and configure cookie management.

Compatibility

Which browsers are supported?

  • ✅ Chrome 88+ (Full support)
  • ✅ Firefox 91+ (Full compatibility)
  • ✅ Brave (Chromium-based)
  • ✅ Edge (Chromium-based)
  • ❌ Safari (Not currently supported)

Does it work on mobile?

Not yet. Mobile browser extensions have limited capabilities. We're exploring mobile app solutions for future releases.

Can I use it with other privacy extensions?

Generally yes, but some conflicts may occur. We recommend disabling similar extensions to avoid conflicts and maximize performance.

Does it work on all websites?

Yes, the extension works on all websites. Some sites with strict Content Security Policies may limit certain protections.

Technical

How do I install it manually?

See our User Guide for detailed installation steps for Chrome and Firefox.

The extension isn't working properly

Try these steps:

  1. Ensure latest version is installed
  2. Disable other privacy/security extensions temporarily
  3. Clear browser cache and restart
  4. Check browser console for errors
