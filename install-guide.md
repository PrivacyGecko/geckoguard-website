# Installation Guide - Gecko Guard Beta

## Quick Installation

### Chrome & Chromium Browsers (Chrome, Brave, Edge)

#### Option 1: Chrome Web Store (Coming Soon)
1. Visit [Chrome Web Store](#) 
2. Click "Add to Chrome"
3. Confirm permissions
4. Extension installed! Look for the Gecko Guard icon in your toolbar

#### Option 2: Direct Install (Available Now)
1. Download `pricko-gecko-guard-chrome.zip`
2. Unzip the file to a folder
3. Open Chrome ‚Üí Menu ‚Üí More Tools ‚Üí Extensions
4. Enable "Developer mode" (top right)
5. Click "Load unpacked" 
6. Select the unzipped folder
7. Extension loaded! Pin it to your toolbar

### Firefox

#### Option 1: Firefox Add-ons (Coming Soon)
1. Visit [Firefox Add-ons](#)
2. Click "Add to Firefox"
3. Confirm permissions
4. Extension installed!

#### Option 2: Direct Install (Available Now)
1. Download `pricko-gecko-guard-firefox.xpi`
2. Open Firefox ‚Üí Menu ‚Üí Add-ons and Themes
3. Click the gear icon ‚Üí "Install Add-on From File"
4. Select the downloaded `.xpi` file
5. Click "Add" to confirm permissions
6. Extension installed!

## Troubleshooting

### Extension Not Working
- **Chrome**: Check if Developer mode is enabled
- **Firefox**: Ensure unsigned extensions are allowed
- **All**: Try disabling other privacy extensions temporarily
- **Reset**: Uninstall and reinstall the extension

### Permission Issues
- Grant all requested permissions for full functionality
- Some features require "All sites" access
- Privacy protection needs these permissions to work

### Performance Issues
- Extension is optimized for minimal impact
- If experiencing slowness, disable other extensions temporarily
- Report persistent issues to our Discord community
