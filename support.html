Before Contacting Support

Prepare this information to help us assist you faster:

  • Browser name and version (Chrome 120, Firefox 115, etc.)
  • Operating system (Windows 11, macOS 14, Linux Ubuntu, etc.)
  • Extension version (check in browser extensions page)
  • Detailed description of the issue or question
  • Steps to reproduce (if reporting a bug)
  • Screenshots or console errors (if applicable)

Support Hours & Expectations

Support Availability

As an open-source project maintained by volunteers and contributors:

  • Discord: Community support available 24/7, core team typically responds during business hours (US Eastern Time)
  • Email: Monitored Monday-Friday, 9am-5pm ET. Responses within 24-48 hours
  • GitHub Issues: Reviewed regularly, responses within 2-5 business days

Priority Guidelines

  • Critical bugs: Security issues and major functionality breaks - Highest priority
  • Bug reports: Non-critical issues affecting functionality - High priority
  • Feature requests: Enhancement suggestions - Medium priority
  • General questions: Usage help and clarifications - Standard priority
