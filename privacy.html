Gecko Guard is designed with privacy-first principles. We don't track you, collect your data, or send anything to our servers. Everything happens locally on your device.

What Data the Extension Accesses

Local Processing Only

The extension accesses certain data to provide privacy protection:

  • Websites you visit - To apply blocking rules and protections
  • Cookies on websites - To categorize and manage them locally
  • Browser fingerprinting attempts - To detect and block them
  • Network requests - To block trackers and unwanted content

Data Storage

All data is stored locally on your device:

  • Privacy settings and preferences
  • Statistics about blocked content
  • Custom rules and whitelists
  • No data is transmitted to our servers

Permissions Explained

Required Browser Permissions

  • activeTab - Read current tab URL to apply protections
  • declarativeNetRequest - Block unwanted network requests
  • storage - Save your settings locally
  • scripting - Inject privacy protection scripts

Why We Need These Permissions

Each permission is essential for privacy protection and used exclusively for that purpose. We never use permissions for data collection or tracking.

Third-Party Services

Current Implementation

  • No third-party analytics or tracking services
  • No external API calls for core functionality
  • All features work completely offline
  • No cloud services or remote data storage

Your Control

Settings Management

  • All settings stored locally on your device
  • Export/import your configuration
  • Reset to defaults at any time
  • Completely remove all data by uninstalling

Transparency

  • Open source code on GitHub
  • Open development process
  • Community-driven feature requests

Data Security

Local Security

  • Extension follows browser security best practices
  • No remote code execution
  • Secure local storage encryption
  • Regular security updates

No Data Breaches Possible

Since we don't collect or store your data on our servers, there's nothing to breach. Your data stays on your device.
