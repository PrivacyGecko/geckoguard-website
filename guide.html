User Guide

Everything you need to know about Gecko Guard

Quick Start

Chrome & Chromium Browsers (Chrome, Brave, Edge)

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Confirm permissions when prompted
  4. Look for the Gecko Guard icon in your toolbar
  5. Click the icon to start protecting your privacy!

Firefox

  1. Visit the Firefox Add-ons Store
  2. Click "Add to Firefox"
  3. Confirm permissions when prompted
  4. Extension installed! Find it in your toolbar
Tip: Pin the extension to your toolbar for easy access. Right-click the extension icon and select "Pin".

Main Features

Block Trackers

Automatically blocks ads, analytics, and social media trackers from following you across the web. This feature stops companies from building profiles about your browsing habits.

Spoof Fingerprint

Randomizes your browser fingerprint to prevent device tracking. Websites can't identify you based on your unique browser configuration.

Block Tracking Cookies

Automatically deletes cookies that track you across sites while preserving essential cookies that keep you logged in.

How to Use

Enable Protection

  1. Click the Gecko Guard icon in your browser toolbar
  2. Toggle the features you want to enable:
    • Block Trackers: Stops ads and analytics
    • Spoof Fingerprint: Randomizes browser identity
    • Block Tracking Cookies: Deletes tracking cookies
  3. Protection starts immediately!

Quick Enable All Protection

Click the green "Enable Basic Protection" button to turn on all three main features at once.

View Statistics

The popup shows real-time stats:

  • Trackers: Number of trackers blocked
  • Cookies: Tracking cookies deleted
  • Spoofed: Fingerprinting attempts blocked

Troubleshooting

Site Not Working Properly

If a website breaks after enabling Gecko Guard:

  1. Click the extension icon
  2. Try disabling "Block Trackers" first
  3. Refresh the page
  4. If still broken, try disabling other features one by one
Common Fix: Some sites require tracking cookies to function. If a site won't work, temporarily disable "Block Tracking Cookies" for that site.

Extension Not Working

  • Check permissions: Ensure you granted all requested permissions
  • Update browser: Make sure you're on the latest browser version
  • Reload extension: Go to browser extensions page and click "Reload"
  • Reinstall: Remove and reinstall the extension if issues persist

Performance Issues

Gecko Guard is optimized for minimal impact, but if you experience slowness:

  • Try disabling other privacy extensions temporarily
  • Clear your browser cache and restart
  • Report persistent issues on our GitHub

Understanding Your Privacy Score

Gecko Guard displays your protection status:

  • Protected: All features enabled - maximum privacy
  • Partial: Some features disabled - moderate privacy
  • Disabled: All features off - no privacy protection

Best Practices

Recommended Setup

For best privacy with minimal site breakage, enable all three main features. Most websites will work fine with these settings.

Browsing Tips

The extension works automatically in the background. Just browse normally and check your stats periodically to see what's being blocked.

Keep Updated

Browser stores update the extension automatically. Check for updates regularly to get the latest blocking rules and features.

Privacy Tips

  • Use Gecko Guard alongside other privacy tools like a VPN for maximum protection
  • Regularly clear your browser cache and cookies
  • Use private/incognito mode for sensitive browsing
  • Review and limit browser permissions for all extensions
